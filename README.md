# Cataclysm-Starscorched
Cataclysm: Starscorched or just Cataclysm is a Minecraft mod that intends to add several new weapons, blocks, mobs, mechanics and bosses. 

# How this was made
This project was made in the Fabric modding engine using Java

# Extra Info
Weapons and mobs are inspired by other mods such as: Celestisynth, Simply Swords, Marium's Soulslike Weaponry, Reliquary and Scape and Run: Parasites.
In NO way does this relate to L_Ender's Cataclysm, a mod with a similar name to this one.
